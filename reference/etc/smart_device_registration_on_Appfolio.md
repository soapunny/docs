# __3. When you bought the devices and want to put them in the warehouse__

## __3-1. Click - Add Fixed Assets__
<img src="https://soap-docs.s3.us-east-2.amazonaws.com/photos/appfolio_addFixedAssets.png">
<br/><br/>


## __3-2. Register - New Fixed Assets__
<img src="https://soap-docs.s3.us-east-2.amazonaws.com/photos/appfolio_newFixedAssets.png">

```
    // You may refer to an existing registered device of the same model on Appfolio.
    1. Choose the device type from the list. (ex. Alexa - echo dot)
    2. Type the maker (ex. Amazon)
    3. Type the model name
    4. Type the model number
    5. Type the year of the device.
    6. Type the serial number of the device.
    7. Type the original price(not discount price).
    8. Choose In storage.
    9. Choose OFFICE WAREHOUSE - SMARTLAND - 67 Alpha ...
    10. The date you bought the device.
    11. Just leave it.
    
    //Click the 'Save' button after change.
```
<br/><br/>


## __3-3. Write the Asset ID on the box of the device__
+ Right after you push the 'Save' button the webpage will give you Asset ID.
+ Write the Asset ID in several places on the box.
+ Now you are all done.
<br/><br/><br/><br/>


# __4. If you want to register the non-registered devices(No AssetID on the box) after installation__

## __4-1. Click - Add Fixed Assets__
<img src="https://soap-docs.s3.us-east-2.amazonaws.com/photos/appfolio_addFixedAssets.png">
<br/><br/>

## __4-2. Register - New Fixed Assets__
<img src="https://soap-docs.s3.us-east-2.amazonaws.com/photos/appfolio_newFixedAssets.png">

```
    // You may refer to an existing registered device of the same model on Appfolio.
    1. Choose the device type from the list. (ex. Alexa - echo dot)
    2. Type the maker (ex. Amazon)
    3. Type the model name
    4. Type the model number
    5. Type the year of the device.
    6. Type the serial number of the device.
    7. Type the original price(not discount price).
    8. Choose Installed.
    9. Choose the unit or property that you install the device.
    10. The date you installed the device.
    11. Just leave it.

    //Click the 'Save' button after change.
```
<br/><br/><br/><br/>


# __5. In case you have to update the status of registered devices(AssetID is marked on the box) after installation__

## __5-1. Search - Fixed Assets__
<img src="https://soap-docs.s3.us-east-2.amazonaws.com/photos/appfolio_fixedAssets-search.png">

+ You can search it with
    - Property or Unit name
    - Status
    - Type

<br/><br/>

## __5-2. Search - Fixed Assets__
<img src="https://soap-docs.s3.us-east-2.amazonaws.com/photos/appfolio_fixedAssets-search-list.png">

+ If you searched correctly, a list of equipment will be displayed below, and you can select the Asset ID that matches the AssetID on your equipment box.

## __5-3. Edit - Fixed Assets__
<img src="https://soap-docs.s3.us-east-2.amazonaws.com/photos/appfolio_fixedAssets-edit.png">

+ Click The 'Edit' buttton.
